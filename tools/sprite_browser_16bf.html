Oryx 16-bit Fantasy — Creature Sprite Browser

Blue-outlined frame = frame A (standing, what the game loads by default).  Right frame = frame B (action pose).  Click any cell to copy the YAML key to clipboard.

Usage: find the sprite you want → put the key: N value in config/tilesets/16bit_fantasy.yaml under entities:
Batch-of-18 layout: within each row, all 18 frame A sprites come first, then all 18 frame B sprites (18 files later).
frameA(packed) = packed + 18 × ⌊(packed−1) / 18⌋  —  frameB = frameA + 18
YAML key = frameA file number (with frame_stride: 1, the key IS the file number loaded).
Note: some names genuinely repeat — the Oryx sheet has 3 sets of human NPC palettes.
